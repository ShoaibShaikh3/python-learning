# Python Conditional Expressions Practice

This repository contains beginner-friendly Python programs based on **conditional statements (if, elif, else)**.

## 📌 Topics Covered
- if statement
- if-else statement
- if-elif-else ladder
- comparison operators
- logical operators

---

## 📁 Programs Included

### 1. Even or Odd Checker
Checks whether a number is even or odd.

### 2. Voting Eligibility Checker
Checks if a person is eligible to vote based on age.

### 3. Grade Calculator
Assigns grades based on marks.

### 4. Leap Year Checker
Determines whether a year is a leap year.

### 5. Simple Login System
Basic authentication using username and password.

### 6. Spam or not message System
It wil check Message is spam or not.
