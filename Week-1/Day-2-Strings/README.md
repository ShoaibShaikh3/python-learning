## Programs

### 1. Reverse String

Reverses a given string using string slicing.

### 2. Palindrome String Checker

Checks whether a string reads the same forward and backward.

### 3. Anagram Checker

Checks whether two strings contain the same characters in a different order.

### 4. Count Vowels

Counts the total number of vowels present in a string.

### 5. Character Frequency Counter

Counts the occurrence of each character in a string.
